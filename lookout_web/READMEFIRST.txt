To preview all webpages properly, please install the Live Server web extension:
Chrome: https://chrome.google.com/webstore/detail/live-server-web-extension/fiegdmejfepffgpnejdinekhfieaogmj?hl=en-US
Firefox: https://addons.mozilla.org/en-US/firefox/addon/live-server-web-extension/?src=search

If you have Visual Studio Code and have the Live Server extension installed, you can do view that way as well.
